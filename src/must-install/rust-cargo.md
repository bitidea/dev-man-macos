## y2z/monolith

[https://github.com/y2z/monolith](https://github.com/y2z/monolith)

将完整网页保存为单个 HTML 文件

### 安装

```bash
cargo install monolith
```

### 用法

```bash
monolith https://dev-man-macos.d8s.fun -o a.html
```
